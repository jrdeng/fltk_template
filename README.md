# fltk_template

A project template using [fltk-rs](https://github.com/fltk-rs/fltk-rs) and [fl2rust](https://github.com/fltk-rs/fl2rust).

## dependencies

    cargo install fltk-fluid

## usage

Modify UI by using fluid:

    fluid ui/ui.fl

Add event handling code in `src/main.rs`.

See also `build.rs`.
